Eclipse で学習するために：

Eclipse を用いて演習を行う時に、各ディレクトリごとにプロジェクトを作る
と手間がかかるため、全体を一つのプロジェクトとして Eclipse に読み込むの
が現実的です。しかし、そうすると、全体が一つのデフォルトのパッケージに
入ることになり、名前の衝突（たとえば、gui と event の両方に Hello.java
があります）が生じます。

このディレクトリに置かれているのは、この問題を解消するために、applet 以
外の各ディレクトリをパッケージとしたライブラリです。教科書の記述に対し
て、各ソースファイルの先頭に、package ***; の1行が追加されています。ま
た、net ディレクトリの、KaiwaServer.java, KaiwaMThread.java の Counsel
というクラスを、Counsel.java に独立させました。

.gif, .jpg, .txt などの拡張子をもつ画像ファイルやテキストファイルは、プ
ログラムやEclipseの実行の構成に何の変更もなく読み込めるように、個々のディ
レクトリには置かずに、この Top のディレクトリに置いています。

Eclipse への読み込み方：

   この教科書のためのプロジェクトを作成します。「新規 Java プロジェクト」
  を選び、「java-everyone」などの適当なプロジェクト名で、「内容」とし
  て「学部ソースからプロジェクトを作成」を選択し、このファイルの置いて
  あるディレクトリを指定します。
